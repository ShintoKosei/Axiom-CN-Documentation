# 强制放置

**强制放置**功能可以绕过正常的放置限制。

比如：

- 通常需要放置在固体方块上的方块可放置在空中，或是随意放置到其他位置。
- 当有实体进行遮挡时，您也可以正常放置方块。（玩家遮挡也可以放置）
- 农作物可以不再放置于耕地，而是随意放置到其他位置。
- 无论光照水平如何，都可以进行方块的放置。
- 以及更多！

> **强制放置**功能最好与[冻结更新](noupdates.md)功能一起使用。以确保强制放置的方块不会被方块更新所影响。在有时，您可以还需要使用`/gamerule randomTickSpeed 0`指令来禁用随机更新。
