# 堆叠工具

堆叠工具可以让你复制选取中的所有方块，并按照您的朝向进行堆叠。

## 选择区域

请查看[Move](move.md)

## 堆叠

当您在选择完区域之后，向某个朝向进行滑动鼠标滚轮。堆叠工具会复制并进行首尾相连的堆叠工作。

向单一的方向进行滑动鼠标滚轮则是一维堆叠，而当您向另一个方向开始滑动鼠标滚轮时则会从一维堆叠变为二维堆叠。同理，当您朝第三个方向进行滚动时，则从二维堆叠变成了三维堆叠。

当您要确定堆叠时，只需按下右键即可。

若您需要进行撤回/复原的操作，则可以使用Ctrl+Z/Ctrl+Y。
