# 延展工具

在三维的方向上进行方块的延展工作。

## 选择区域
请查看[Move](move.md)

## 延展

向一个方向进行滑动鼠标滚轮来移动。

当您要确定延展时，只需按下右键即可。

若您需要进行撤回/复原的操作，则可以使用Ctrl+Z/Ctrl+Y。
